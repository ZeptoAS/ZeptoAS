Zepto Analytics

Zepto Analytics is a web-application for managing and visualizing sensor data stored in a database

Currently it supports collecting and displaying data from the following sources:

  • Sensor data from Zepto Sensors
  • Weather data from Forecastio
  • Energy data from Nordpool
